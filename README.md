CFA Time Value Engine

**QuantTVM Engine** is a professional-grade **Time Value of Money (TVM)** analytics engine designed for **CFA curriculum applications** and **quantitative finance workflows**. It provides robust computations for lump sums, ordinary annuities, annuities due, interest rate solving, period calculations, and full cash-flow schedules.

This engine bridges financial theory with practical quantitative implementation, offering both accuracy and extensibility for learners, developers, and finance professionals.

---

## Features

- Compute **Future Value (FV)** and **Present Value (PV)** for lump sums and annuities  
- Solve for **Periodic Payment (PMT)** given PV or FV  
- Determine **Number of Periods (N)** or **Interest Rate (R)** using numerical methods  
- Support for **Ordinary Annuities** and **Annuities Due**  
- Generate **full amortization/investment schedules** with period-by-period breakdowns  
- Interactive **command-line interface** for quick calculations  
- Optionally **plot balances and interest/principal over time**
